## How to Capture

See detailed guide: [`../SCREENSHOT_GUIDE.md`](../SCREENSHOT_GUIDE.md)

### Quick Steps:
1. Run application: `python app.py`
2. Press `Win + Shift + S` to take screenshot
3. Save as PNG in this folder
4. Recommended size: 1920x1080 or 1280x720

## File Specifications

- **Format:** PNG
- **Max Size:** 500KB per image
- **Resolution:** 1280x720 or higher
- **Naming:** lowercase with hyphens

## Compression

Use online tools to compress:
- https://tinypng.com/
